# B2LoRa_M

This repository contains the MCU implementation of B²LoRa-M on an STM32 platform. B²LoRa-M is a blind coherent combining system designed to enhance LoRa downlink reliability in satellite-IoT networks. The codebase focuses on the core on-device processing pipeline, showing how B²LoRa-M can run on a low-power MCU and process real-world satellite I/Q traces under practical resource constraints.
